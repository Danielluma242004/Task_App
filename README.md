## TASK APP

## Descripcion
Esta es una aplicación web desarrollada en el FrameWork de Django, utlizando las siguientes tecnologías:
-Lenguaje de Programación : JavaScript
-FrontEnd : ReactJS, Tailwindcss
-BackEnd : Django, Axios
-Database: sqlite3
